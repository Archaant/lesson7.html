Comments

With comments you can place notifications and reminders in your HTML code. Example 1:

                             <!-- This is a comment --> 

                            This is a paragraph.

                             <!-- This is also a comment. Remember to add more information here -->

Comments are not displayed by the browser, but they can help document your HTML source code.
Comments can be used to hide content. This can be helpful if you hide content temporarily.
You can also hide more than one line.
Everything between the <!-- and the --> will be hidden from the display.
Comments are also great for debugging HTML, because you can comment out HTML lines of code, one at a time, to search for errors.

This is how example 1 really looks like in your browser.
